# Parcours C++ 42

Ce dépôt regroupe mes projets du cursus C++ 42. Chaque module est isolé dans son propre dossier, avec ses exercices, son Makefile et, lorsque c'est utile, une documentation détaillée au niveau du module ou de l'exercice.

L'objectif de ce README est de servir de point d'entrée rapide. Si vous cherchez le détail d'une implémentation, ouvrez le README du module concerné.

## Navigation

- [CPP00](CPP00/README.md) - Premiers pas avec C++, classes simples et I/O.
- [CPP01](CPP01/README.md) - Pointeurs, références, allocation dynamique et classes utilitaires.
- [CPP02](CPP02/README.md) - Forme canonique, surcharge d'opérateurs et nombre à virgule fixe.
- [CPP03](CPP03/README.md) - Héritage et hiérarchies de classes.
- [CPP04](CPP04/README.md) - Polymorphisme, classes abstraites et copies profondes.
- [CPP05](CPP05/README.md) - Exceptions et gestion d'un flux de validation métier.
- [CPP06](CPP06/README.md) - Casts explicites et conversions de type.
- [CPP07](CPP07/README.md) - Templates, fonctions génériques et classe template.
- [CPP08](CPP08/README.md) - STL, itérateurs et algorithmes.
- [CPP09](CPP09/README.md) - Conteneurs STL et structures de données classiques.

## Organisation du dépôt

Chaque dossier `CPPxx` contient en général :

- un dossier par exercice (`ex00`, `ex01`, ...)
- un `Makefile` par exercice, pour une compilation indépendante
- des fichiers `include/`, `includes/`, `src/` ou équivalents selon le module
- une documentation locale quand l'exercice mérite un contexte supplémentaire

## Lecture recommandée

Si vous voulez aller à l'essentiel, commencez par les README des modules 08 et 09, qui sont les plus détaillés. Les nouveaux README des modules 00 à 07 suivent le même principe: un résumé du module, puis un détail exercice par exercice.
